# Web Portfolio App

This is a personal web portfolio application built with React and Vite. It showcases various sections including an introduction, skills, projects, and contact information, all wrapped in a modern, responsive design with dark and light theme support.

## Features

- **Hero Section:** Introduction and landing area with engaging visuals.
- **Tech Section:** Highlights the technologies and skills used.
- **Projects Section:** Displays portfolio projects with images and descriptions.
- **Contact Section:** Provides a way to get in touch.
- **Footer:** Contains additional links or information.
- **Dark/Light Theme Toggle:** Smooth transition between dark and light modes for better user experience.

## Technology Stack

- React 19
- Vite
- TailwindCSS
- Framer Motion (for animations)
- React Icons

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

- To start the development server with hot module replacement:
  ```bash
  npm run dev
  ```
- To build the project for production:
  ```bash
  npm run build
  ```
- To preview the production build locally:
  ```bash
  npm run preview
  ```

## Project Structure

- `src/components/` - React components for different sections of the portfolio.
- `src/assets/` - Images, fonts, and other static assets.
- `src/App.jsx` - Main application component organizing the layout.
- `src/index.css` - Global styles including TailwindCSS imports.
